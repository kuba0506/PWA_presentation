Different type of apps.

  • Native app A native app is written in the programming language specific for a platform, the most popular being Objective-C or Swift for Apple devices and Java for Android devices.
  • Hybrid app A hybrid app is a web app that translates to native code on a platform like iPhone or Android. A hybrid app uses a browser view and hooks to allow your web app to access features on your mobile device like Push Notifications, Contacts, or Offline Data Storage.

    Hybrid app development tools:
    • PhoneGap
    • Rubymotion
  • Web app A web app is an app written in web code (HTML, CSS, JS) that is similar to a website but more interactive so it feels like an app on mobile devices.

What is Progressive Web Apps?  1/3

Progressive web apps are web applications which are designed to work even more seamlessly on mobile devices than native mobile apps.
Most web traffic comes from mobile devices, and that users install between 0–3 new apps per month, on average. That means that people aren’t spending a lot of time looking for new apps to try out in the app store, but they are spending lots of time on the web, where they might discover and use your app.

What is Progressive Web Apps?  2/3

  • start out just like any other web app (splash screen, full screen)
  • have its own home screen icon - no URL bar and no web navigation UI
  • PWA’s can also benefit from push notifications, like native apps
  • may sync in background
  • cache necessary data
  • PWAs works cross-platform

What is Progressive Web Apps?  3/3

Besides PWAs are:

  • Reliable - Load instantly and never show the downasaur, even in uncertain network conditions.
  • Fast - Respond quickly to user interactions with silky smooth animations and no janky scrolling.
  • Engaging - Feel like a natural app on the device, with an immersive user experience.

Progressive Web App Checklist 

  • Site is served over HTTPS
  • Pages are responsive on tablets & mobile devices
  • All app URLs load while offline (Service Workers, Cache API)
  • First load fast even on 3G
  • Must supply its own navigation (back button)
  • Valid manifest with required properties (Home icon)
  • Optional: Push Notifications
  • Optional: Background Sync

Testing PWA – Lighthouse

Lighthouse is an open-source, automated tool for improving the quality of web pages. You can run it against any web page, public or requiring authentication. It has audits for performance, accessibility, progressive web apps, and more.
